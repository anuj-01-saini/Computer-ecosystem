# PERSONAL TRAVEL GUIDER
This is a demo practise repository which is having mini travel guider web app

A Smart Travel Guide Web Application helps travelers explore destinations by providing
useful information such as place description, translation of foreign languages, and sentiment
analysis of travel reviews.
Microsoft Azure provides cloud-based Artificial Intelligence services that can be integrated
into web applications. In this experiment, three Azure services are used:
Azure Vision API is used to analyze travel images. It identifies landmarks, objects, and
generates descriptions of images.
Azure Translator API is used to translate foreign language text into English. This helps
travelers understand signboards, menus, and travel guides written in different languages.
Azure Text Analytics API is used to perform sentiment analysis on travel reviews. It
identifies whether reviews are positive, negative, or neutral.
The Smart Travel Guide Web App integrates all three services into a single interface. Users
can upload travel images, input travel text, and analyze reviews. The app processes data
using Azure services and displays results.
This system improves travel experience by providing intelligent recommendations and
multilingual support.
